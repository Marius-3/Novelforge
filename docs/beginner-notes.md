# 新手说明

## 什么是后端？

后端就是“真正干活的程序”。

在 NovelForge 中，后端负责：

- 接收你的小说创意。
- 调用 AI 模型。
- 检查 AI 返回的数据格式。
- 保存小说项目文件。
- 给前端提供接口。

## 什么是前端？

前端就是“你看到和点击的网页界面”。

例如：

- 新建小说按钮。
- 章节编辑器。
- 人物设定面板。
- 世界观设定页面。

## 什么是 API？

API 是前端和后端之间的通道。

例如前端点击“生成书级方案”后，会请求后端的这个地址：

```text
POST /api/director/book-plan
```

后端收到请求后调用 AI，再把结果返回给前端。

## 为什么第一步先做后端？

因为 AI 小说工作台的核心不是漂亮界面，而是：

1. 能不能稳定调用 AI。
2. 能不能生成结构化设定。
3. 能不能保存和管理长期设定。
4. 能不能避免长篇小说前后矛盾。

所以先做后端，再做前端。
